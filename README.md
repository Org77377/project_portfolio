# 🌐 Omkar RG - Portfolio Website

> 🔗 **Live Demo:** [portfolio-omkarrg.vercel.app](https://portfolio-omkarrg.vercel.app)

- Fully responsive design
- Clean and minimal layout
- Sections for:
  - About Me
  - Projects
  - Skills
  - Contact
